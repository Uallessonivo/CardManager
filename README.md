# Card Manager

Card Manager é uma aplicação para gerenciar cartões.

# Tecnologias

* .NET 7
* PostgreSQL
* Entity Framework Core
* FluentValidation
* AutoMapper


# Arquitetura

A aplicação segue a arquitetura limpa (Clean Architecture), onde temos as seguintes camadas:

* Presentation
* Application
* Domain
* Infrastructure
